# Copilot Harness - Promotional Website

Simple single-page React (Vite) site with Tailwind via CDN showcasing Copilot Harness.

## Quick Start
1. Ensure Node.js >=18, npm, and gh CLI installed
2. Run: `./init.sh`
3. Open the printed localhost URL (default http://localhost:5173)

## Tech Stack
- Frontend: React + Vite
- Styling: Tailwind CSS via CDN
- Icons: Heroicons/SVG
- Animations: CSS transitions + simple scroll effects

## Structure
- frontend/: Vite React app
- docs/: Additional documentation
- backend/: (unused; static site)
